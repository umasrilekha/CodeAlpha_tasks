# CodeAlpha Stock Portfolio Tracker

This is a simple Python-based Stock Portfolio Tracker developed as part of the CodeAlpha Python Programming Internship.

## Project Description
The program allows users to enter stock symbols and quantities. It calculates the total investment value using predefined stock prices and displays a summary. The result is also saved to a text file.

## Features
- Predefined stock prices using dictionary
- User input for stock name and quantity
- Total investment calculation
- Summary display
- Saves output to a text file

## Technologies Used
- Python
- Dictionary
- File Handling

## How to Run
1. Ensure Python is installed
2. Open terminal in project folder
3. Run: python app.py
